接入到 NotionNext 主题

step · by · step

1. 把文件放进主题目录

把项目里的 theme-assets/ 整个文件夹拷贝到你 fork 的仓库下，放在主题的 public 静态目录：

# 在你 fork 的仓库根目录
mkdir -p public/images/logo
cp theme-assets/*.svg  public/images/logo/
cp theme-assets/*.png  public/images/logo/
# favicon 系列单独放到根 public 方便引用
cp theme-assets/favicon.svg            public/favicon.svg
cp theme-assets/favicon-32.png         public/favicon-32.png
cp theme-assets/favicon-16.png         public/favicon-16.png
cp theme-assets/apple-touch-icon.png   public/apple-touch-icon.png
cp theme-assets/maskable-512.png       public/maskable-512.png

2. 在 blog.config.js 改站点配置

module.exports = {
  // ...
  BLOG_FAVICON: '/favicon.svg',
  BLOG_AVATAR:  '/images/logo/logo-256.png',
  BLOG_LOGO:    '/images/logo/logo-mark.svg',      // 导航栏 mark
  BLOG_LOGO_DARK: '/images/logo/logo-mark-dark.svg',
  HOME_BANNER_IMAGE: '/images/logo/logo-lockup.svg', // 可选
  SEO_DEFAULT_IMAGE: '/images/logo/maskable-512.png',
  // ...
}

3. 在 _document.js 或 Head 注入完整 favicon 族

<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png" />
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png" />
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
<link rel="mask-icon" href="/favicon.svg" color="#ffc828" />
<meta name="theme-color" content="#ffc828" />

4. 在 xiyu 主题 SiteNav 组件里替换 brand 块

// themes/xiyu/components/SiteNav.js
import Image from 'next/image'

<a href="/" className="brand">
  <Image src="/images/logo/logo-mark.svg" width={36} height={36} alt="xiyu" priority />
  <span className="brand-mark">xiyu<span className="brand-dot" /></span>
  <span className="brand-tag">long · bitcoin</span>
</a>

5. （可选）OG image 与 Notion 头像

登录 Notion，把关于页那张头像 block 也换成 logo-256.png，这样所有入口视觉统一。Vercel 部署后 OG 预览会自动用 SEO_DEFAULT_IMAGE。

6. 验证清单

  • 浏览器 Tab 图标 · 16px 发髻 + 眼镜仍可辨认
  • iOS "添加到主屏" · 黄底圆角无白边
  • 深色模式 Nav · 自动切到 logo-mark-dark.svg
  • Twitter / 微信分享 · 卡片能抓到 512 px 图
